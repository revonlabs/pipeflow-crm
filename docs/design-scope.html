06 — Catalogo de Animacoes
Especificacoes Tecnicas

Todas as animacoes usam apenas CSS @keyframes e transitions. Propriedades animadas: transform e opacity (GPU-accelerated, sem reflow).

orbFloat
@keyframes 10s ease-in-out infinite

Movimento organico dos gradient orbs. Combina translate + scale para simular flutuacao natural.

waveScroll
@keyframes 12-18s linear infinite

Translada SVG wave horizontalmente em loop. 2 camadas com velocidades diferentes criam profundidade.

flowPulse
@keyframes 3s ease-in-out infinite

Pulsacao da linha da logo. ScaleX 0.5→1 + opacity 0.3→1. Sutil e elegante.

caseFloat
@keyframes 3.5s ease-in-out infinite

Flutuacao dos case cards. TranslateY 0→-6px. Delays alternados entre cards.

fadeSlideUp
@keyframes 0.4s ease-out forwards

Entrada de elementos. Opacity 0→1 + translateY(15px→0). Triggered por Intersection Observer.

counterUp
JS vanilla 1.5s ease-out

Contagem animada de numeros nos cases. ~20 linhas de JS. RequestAnimationFrame + Intersection Observer.

Hover: Cards
transition 0.3s ease

TranslateY(-4px) + border-color accent + box-shadow glow. Aplicado em cards de metricas, features e cases.

Hover: CTA Button
transition 0.2-0.3s ease

TranslateY(-2px) + box-shadow 30px glow accent (30% opacidade). Efeito de "acender".

08 — Stack Tecnico
Decisoes de Implementacao

Abordagem "Flow Elegante" — maximo impacto visual com minimo impacto tecnico.

AspectoDecisaoImpacto
Animacoes CSS @keyframes + transition 0kb
Scroll Detection Intersection Observer (API nativa do browser) 0kb
Counter Animation JS vanilla (~20 linhas, requestAnimationFrame) ~0.5kb
Ondas / Waves SVG inline estatico com CSS animation ~1kb SVG
Glassmorphism backdrop-filter: blur() CSS nativo 0kb
Logo Componente React com SVG inline ~0.3kb
Novas dependencias Nenhuma 0 libs
Bundle size total adicionado CSS + SVG + JS vanilla ~2kb

Performance

GPU Accelerated

Todas as animacoes usam transform e opacity — propriedades composited pela GPU sem causar layout/paint.

60fps Target

Nenhuma animacao causa reflow. Tudo roda no compositor thread do browser. Testavel via DevTools Performance tab.

Reduced Motion

Media query prefers-reduced-motion desabilita todas as animacoes para usuarios com sensibilidade a movimento.

09 — Escopo e Limites
O que Muda vs. O que Nao Muda

Dentro do Escopo

  • Hero da LP com gradientes animados e waves
  • Nova secao de Cases de Sucesso com metricas
  • Nova logo tipografica "PipeFlow"
  • SVG waves entre secoes da LP
  • Animacoes de entrada (fade+slide) nas features
  • Glassmorphism nos cards do dashboard
  • Hover effects aprimorados em cards/botoes
  • Animacao stagger nos cards de metricas
  • Animacao stagger nas colunas do kanban
  • Hover colorido nos deal cards do pipeline
  • Gradiente sutil no header do dashboard
  • Hover suave nas rows da tabela de leads
  • Animacao de entrada em todas as paginas
  • Gradiente no item ativo da sidebar
  • Counter animado nos numeros dos cases
  • Suporte a prefers-reduced-motion

Fora do Escopo

  • Logica de negocio / hooks / API routes
  • Autenticacao e middleware
  • Integracao Stripe (pagamentos)
  • CRUD de leads, deals, membros
  • Drag-and-drop do pipeline (dnd-kit)
  • Estrutura de rotas e navegacao
  • Schema do banco de dados (Supabase)
  • Adicao de novas dependencias ao package.json
  • Mudancas no sistema de cores existente
  • Mudancas nas fontes existentes
  • Alteracoes no .env ou configuracoes
